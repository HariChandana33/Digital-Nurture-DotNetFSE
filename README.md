# Upskill Training Assignments

This repository contains my solutions for the upskilling assignments. The project is divided into the following modules:

* **C# and ADO.NET**: Contains 30 exercises covering C# fundamentals, Object-Oriented Programming (OOP) basics, LINQ, and performing CRUD operations using ADO.NET[cite: 1].
* **ANSI SQL Using MySQL**: Contains 25 queries based on an Event Management database schema, including users, events, sessions, registrations, and feedback[cite: 2].
* **HTML5**: Contains 10 exercises focused on building a lightweight, browser-based portal for a local community event using semantic HTML, forms, and geolocation[cite: 3].
* **jQuery**: Contains 6 exercises focusing on library inclusion, DOM manipulation, and working with mouse and keyboard events[cite: 4].
